This week we started our electical engineering unit. We learned about the different kinds of electrical components like resistors, capacitors, diodes, power sources, etc. We also started designing circuits in a program called TinkerCAD. We've used TinkerCAD in the passed for the Mechanical Engineering unit, but it also has a circuit designer. We have to drag and drop components together or connect them with the built in wires. Once we built the circuits on a breadboard (a way to distrubite power and build circuits for prototyping) we could then press the simulation button to see how the electricity would run through the circuit and whether or not it would blow up components or give possible damage to anything. This is what we did this week, and I very much enjoyed learning the basics. Next week we are supposed to start learning the more indepth parts of electrical engineering.







11/6/23 - 11/10/23

This week we continued our electrical engineering unit. We built circuits on breadboards that lit up LEDs (Light Emitting Diodes). At first, we used an Arduino, a small computer, just as power for the circuits, but later in the week we used it to program things for the LEDs to do. We had to connect the power,resistors, LEDs, and ground in a certain order for things to work properly. We were able to make the LEDs flash using the Arduino, create a sequence of lights, and even make it fade on and off. The Arduino programming IDE used C which is a quite a old programming language. It was interesting to see the differences between a programming language that was developed a long time ago and things that were created recently. Overall, this week was very fun and I am excited to see what we are going to do next.
