2/26/24

These past few weeks, we have been building an Arduino car that we programmed to avoid obstacles. In the end, we got the C++ code to work and the car was able to avoid obstacles in a relatively good way. You can look at some of the previous updates to learn more about the process of making the arduino car. Overall though, my groupmate and I used Fusion 360 to model and CAD the car. This allowed us to make an extremely accurate model of the car that we wanted to create. We were able to use the TinkerCad circuit designer to create a test some code before we started building the actual car. This allowed use to figure out waht resources that we needed to finish the car. Once we had the design for everything figured out, we 3d printed the model that we had previously designed. 3d printing the design, allowed to create a 1:1 recreation of what we designed on the computer. Using this 3d print, we were able to start building the electronics on top of it. The electronics that we used include 2 DC motors, an Arduino Uno, a L293D motor driver IC, a prototyping breadboard, an ultrasonic distance sensor, and lots of jumper cables for the breadboard. The breadboard acted as a way to connect all of the compents together. Using the arduino, we could hook up to the motor driver which took in a low voltage signal and used it to modulate a higher voltage connection to each of the motors. We were able to hook up the ultrasonic distance sensor to the Arduino, this allowed to know how far from an object we were. Using this information, we could figure out when to turn by comaring the distance detected by the sensor to a set limit that we created. For example, if the distance was closer than 5 centimeters, the car backed up, if it was closer than 15 centimeters it turned left, otherwise, the car turned left. Using all of these thing combined, we were able to create a car that could detect obstacles and avoid them. Overall, I had a lot of fun with this project. Below is a video of the car driving and attempting to avoid obstacles.
